## 1. What this is
A vanilla, no-build static **Misfits Database** — a Fallout/Pip-Boy terminal viewer for SS14 RP
factions. Deployed to GitHub Pages (repo `dr-orac/oracular-db`, live at
https://dr-orac.github.io/oracular-db/, redeploys on push to `main`). One `index.html` + `styles.css` +
`app.js` + self-hosted fonts + `tools/` + `docs/`, no build step. Data: **Google Sheets** rosters + **Google Docs**
lore + the **Misfits wiki** (`wiki.misfitsystems.net`). Only **tribe** + **brotherhood** have real sheet
data today.

## 3. Key architecture (what to know before editing)
- **Hash router**: `#home` · `#<faction>/<section>[/<target>]` · `#wiki/<Page>`. `parseRoute`/`applyRoute`/
  `writeRoute`/`setSection`. Deep-links exist — reuse them for any new nav (T96's global palette can lean
  on this directly).
- **Masthead** = two rows: row-1 primary nav (HOME / FACTION box / WIKI + cog); row-2 `#topnav`
  (Roster / Relations / doc tabs). The **flow-chart connector** hangs off the FACTION box:
  `#topnav::before` (dim full bus), `#topnav::after` (LIT segment — NEW), `.navtab::before` (risers),
  `.navtab::after` (arrowheads), `.faction-box::after` (stem, always lit). All positioned by
  `positionConnector()`; shown only on the wide single-row masthead (`min-width:860px`,
  `:not([data-frame="border"]):not([data-section="home"])`).
- **Bottom console**: `.commandbar` (order:2) holds `.toolbar` (roster search/view/refresh) and `.docbar`
  (doc-find). `.statusbar` (order:3) holds `#sb-crumb` / `.sb-emblem` / `.devsig`. `updateCrumb()` sets the
  breadcrumb (called in `setSection` + end of `loadWiki`).
- **Reader**: `loadWiki`→`renderWiki`→`wikiClean`/`docClean`; `.docreader` grid w/ `--doc-measure`;
  `prepareStackTables`/`evalTableStacking` (T81 reflow); `mergeAdjacentLists` (T24); `leadLabel`/`trailLabel`
  (T80).
- **Theme tokens** (`styles.css` `:root`): `--fg` `#3cff7a`, `--fg-bright` `#b6ffce`, `--fg-dim` `#21954d`,
  `--fg-faint` `#0f3a22`, `--fg-rgb`, `--box-glow`, `--conn-dim/-lit/-glow`, `--sel-line`, `--row-gap`,
  `--bezel-*`, `--frame`, `--edge`. `selfcheck.py` tracks used-vs-defined vars + id refs.
- **Reduced motion**: `mdb-reducemotion` pref (default OFF), `body[data-reducemotion]`, `applyReduceMotion()`,
  `crtRetune()` gated on it. Note: the SANDBOX has system `prefers-reduced-motion: reduce` ON.
